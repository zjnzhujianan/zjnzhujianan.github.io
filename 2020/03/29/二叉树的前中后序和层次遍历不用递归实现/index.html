二叉树的前中后序和层次遍历不用递归实现
发表于 2020-03-29| 更新于 2020-03-29
阅读量:

前序遍历

通过栈来实现前序遍历，前序遍历是根左右的顺序，先让根进栈，然后进入循环（栈不为空），根出栈，赋给动态数组返回，指针指向孩子节点，因为栈先进后出的特性，右孩子先压栈，然后左孩子进栈，不断循环实现根左右的顺序，即前序遍历。

int* preorderTraversal(struct TreeNode* root, int* returnSize){
int res = (int )malloc(sizeof(int)100);
int top = -1;
returnSize = 0;
struct TreeNode *stack = (struct TreeNode**)malloc(sizeof(struct TreeNode)100);
struct TreeNodep = root;
if(root == NULL)
{
return NULL;
}

stack[++top] = p;

while(top != -1)
{
    p = stack[top--];
    res[(*returnSize)++] = p->val;

    if(NULL != p->right)
    {
        stack[++top] = p->right;
    }
    if(NULL != root->left)
    {
        stack[++top] = p->left;
    }
}
return res;

}

中序遍历

同样通过栈来实现中序遍历，中序遍历是左根右的顺序，那么第一个出来的是左子树一直往下的最后一个左子树，所以同样根节点先进栈,然后所有左子树依次进栈，直到找到最后一个左子树，左子树出栈，放入动态数组返回，然后找它的右子树是否存在，以及它的子树是否存在，分两种情况（不多细讲），从而实现左根右的顺序，另外当p返回根节点时，top = -1,所以大循环加了p!= NULL。

int* inorderTraversal(struct TreeNode* root, int* returnSize){
int res = (int )malloc(sizeof(int)100);
int top = -1;
struct TreeNode stack = (struct TreeNode)malloc(sizeof(struct TreeNode)100);
struct TreeNodep = root;
*returnSize = 0;

if(root == NULL)
{
    return NULL;
}

while(p != NULL || top != -1 )
{
    while(p != NULL)
    {
        stack[++top] = p;
        p = p->left;
    }

    p = stack[top--];
    res[(*returnSize)++] = p->val;
    p = p->right;
}

return res;

}

后序遍历

后序遍历是最复杂的一种，我最后通过前序遍历的变式（自右向左）加双栈实现的，就是计算可以得到自右向左的前序遍历就是后序遍历的逆序的情况，因为后序遍历是左右根的顺序，而前序的变式是根右左的顺序，所以总体按照前序遍历来写，但左子树先压栈，右子树再进，最后用两个栈实现顺序逆序，得到后序遍历的结果。

int* postorderTraversal(struct TreeNode* root, int* returnSize){
int res = (int )malloc(sizeof(int)100);
struct TreeNodestack1 = (struct TreeNode)malloc(sizeof(struct TreeNode)100);
struct TreeNodestack2 = (struct TreeNode)malloc(sizeof(struct TreeNode)100);
struct TreeNodep = root;
int top1 = -1;
int top2 = -1;
*returnSize = 0;

if(root == NULL)
{
    return NULL;
}

stack1[++top1] = p;

while(top1 != -1)
{
    p = stack1[top1--];
    stack2[++top2] = p;

    if(p->left != NULL)
    {
        stack1[++top1] = p->left;
    }
    if(p->right != NULL)
    {
        stack1[++top1] = p->right;
    }
}

while(top2 != -1)
{
    p = stack2[top2--];
    res[(*returnSize)++] = p->val;
}

return res;

}

层次遍历

层次遍历是自上而下，自左向右，即先进先出，用队列来实现，先根节点入队然后进入循环，根节点出队输出，然后左右孩子节点依次进队，再左孩子出去，带进它的孩子到队列中，然后有孩子出去，再带它的，最后实现层次遍历。
